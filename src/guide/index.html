レシピを展開中間素材を含む制作工程をツリーで確認

素材から逆引きその素材を使うレシピを検索

アイテムを検索する

  1. 検索欄にアイテム名の一部を入力します。
  2. 候補から作りたいアイテムを選びます。
  3. 候補一覧の隣にレシピツリーが表示されます。
  4. 選んだアイテムのレシピツリー画面へ移動します。

名前が分からない装備は、検索欄右の「▼」からジョブ・レベル・部位で絞り込めます。

装備を条件検索する

検索欄右の「▼」を開き、ジョブ・装備レベル・アイテムレベル・部位を指定します。アイテムレベルには、指定した装備レベルに対応する候補だけが表示されます。「検索結果をお気に入り登録」で候補を一括保存できます。

レシピツリー画面を確認する

選んだアイテムを作るための工程を、完成品から末端素材まで階層表示する画面です。上部では制作個数と「レシピツリー／素材リスト」を切り替えます。ツリー内では各素材の必要数、制作数、制作ジョブ、入手方法を確認できます。

▼／▶中間素材の内訳を開閉します。
素材名その素材を完成品とする個別のレシピツリーを開きます。
📍／📌お気に入りへの登録と解除に使います。
🛒・⏰購入情報または刻限採集情報のフローティングウィンドウを開きます。

必要素材と個数を確認する

「素材リスト」は、レシピツリーの末端素材を種類ごとに合算する画面です。各行に素材名、入手方法、合計必要数を表示し、中間素材には推奨制作順と、どの完成品・中間素材の制作に何個使うかを表示します。

  1. 「個数」で制作したい数を指定します。
  2. 「素材リスト」を選ぶと、末端素材が合算表示されます。
  3. 「レシピツリー」で制作工程へ戻れます。

ツリー内の素材を操作する

枝の開閉「▼／▶」で中間素材の内訳を開閉します。
中間素材名選んだ中間素材のレシピツリーと個数指定を開きます。

🛒 制作・購入とショップ情報

素材を購入して用意する場合に使います。🛒を押すとフローティングウィンドウが開き、購入価格、ショップ名、場所を確認できます。購入指定を有効にすると、その素材より下の制作工程を素材計算から外します。

⏰ 刻限採集

採集できる時間が決まっている素材に⏰が表示されます。⏰を押すとフローティングウィンドウが開き、エオルゼア時間、開始までの時間、採集場所を確認できます。

素材からレシピを逆引きする

素材を選んで「使用先」を押すと、その素材を使うレシピを一覧できます。余った素材の使い道を探すときに便利です。

検索したアイテムと、開いた素材・レシピは「検索履歴」に最大100件保存されます。検索欄の「×」で入力内容を消せます。

お気に入りリストを使う

  1. 検索結果やレシピツリーの📍を押します。
  2. 登録先を選びます。登録時に新しいリストも作れます。
  3. 登録済みの📌を押すと、お気に入りから削除できます。
  4. 「お気に入り」からリストを開き、「素材リストを表示」で全アイテム分を合算します。

お気に入りリストを整理する

リスト右端の「◀」を押すと、✏️・☰・🗑️が表示されます。✏️で名前を変更し、☰をドラッグしてリストの順番を変えます。🗑️を押した場合は確認画面で「はい」を選ぶとリストが削除されます。「▶」で操作を閉じます。

お気に入りリストの拡張機能

「素材リストを表示」の右にある▼を押すと、拡張機能を開きます。

並び替え

リスト内のアイテム順を変えたいときに使います。「並び替え」を有効にし、☰をドラッグします。もう一度「並び替え」を押すと終了します。

個数指定

アイテムごとに作りたい数が異なるときに使います。「個数指定」を有効にして数を入力します。「全て1個」「全て0個」で一括変更できます。「素材リストを表示」で指定数を反映した結果を確認し、「▼製作内容」で対象アイテムを折り畳めます。

どれか1アイテム

登録した候補をすべて作るのではなく、候補のうちどれか1つを作れる素材リストを確認する場合に使います。「どれか1アイテム」を有効にし、比較したい候補をOnにします。「全てOn」「全てOff」で候補を一括変更できます。「素材リストを表示」を押すと、Onにした各候補を1つ作るための必要素材が「もしくは」で区切られます。完成品が直接使う同じ末端素材と、同じ中間素材は、候補間の最大数を1回分だけ表示します。異なる中間素材はそれぞれ表示し、その製作に共通して使う末端素材は合算します。「▼製作内容」で候補表示を折り畳めます。

指輪

指輪を含む素材リストでは、指輪ごとに0・1つ・2つから選びます。選んだ数で必要素材が再計算されます。

複数のお気に入りリストの素材を計算する

計算するリストをチェックする

  1. 「お気に入り」を開き、計算に含めるリスト右端の「◀」を押します。
  2. 表示されたチェックボックスへチェックを入れます。
  3. 同じ操作で、ほかのリストにもチェックを入れます。
  4. 「合算」または「どれか1リスト」を選び、「素材リストを表示」を押します。

1リスト以上をチェックすると、複数リスト用の「拡張機能」が表示され、検索欄と装備検索は上へ折り畳まれます。「？」では2つの計算方法の説明を確認できます。「戻る」を押すと、チェックをすべて外して素材リストを閉じます。

合算

チェックした複数のお気に入りリスト内の全アイテムを製作するために必要な素材を合算して表示します。

どれか1リスト

チェックしたお気に入りリストのうち、どれか1リストをセット数分製作するために必要な素材リストを表示します。チェックしたすべてのリストを製作する素材リストではありません。完成品が直接使う同じ末端素材は各リスト内で合算してから、リスト間の最大数を表示します。同じ中間素材もリスト間の最大数を1回分だけ表示します。異なる中間素材はそれぞれ表示し、その製作に共通して使う末端素材は合算します。

製作内容と指輪の数

「合算」と「どれか1リスト」のどちらでも、「▼製作内容」で対象リストを折り畳めます。指輪を含むリストでは、リストごとに指輪の製作数を0・1つ・2つから変更すると、必要素材が再計算されます。

保存・共有する

お気に入りはブラウザー内に自動保存されます。別端末への移行や仲間との共有には「⚙ 設定」のシェアコードを使います。

  1. 「⚙ 設定」を開き、共有するリストを選びます。
  2. 表示されたコードをコピーします。
  3. 受け取る側はコードを貼り付けて「取り込む」を押します。

出力例：Z00103F0Y320Y1M0Y6D3S526C594C6C563S6D4U4A0Y180Y2X0Y1M2J1D1I1C1J181G1G1E1E2L3H

シェアコード広場

「⚙ 設定」の「シェアコード広場を開く」から、掲載されているお気に入りリストをアイテム名や画像と一緒に確認できます。Discordを開く必要はありません。

  1. 「⚙ 設定」を開き、「シェアコード広場を開く」を押します。
  2. お気に入りへ保存する場合は「シェアコードを取り込む」を押し、リスト名を確認します。
  3. コードだけを別の場所で使う場合は、隣の「シェアコードをコピー」を押します。

小窓・PWAで使う

小窓で開くPCの「⧉ 小窓で開く」で、制作中も置きやすい別ウィンドウを開きます。
ホーム画面に追加Androidはブラウザーのメニューから追加できます。iPhone／iPadはSafariだけがホーム画面へ追加できます。
更新通知新しいアプリが利用可能になると再読み込みボタンが表示されます。

注意事項・設定

  • ブラウザーを閉じただけではお気に入りは消えません。サイトデータの削除、ブラウザーの初期化、端末の故障などに備え、大切なリストはシェアコードを保管してください。
  • 収録アイテムは制作計算に必要なものを中心にしています。
  • 「⚙ 設定」からプライバシーポリシー、LICENSE、Discordの報告・お問い合わせ先を開けます。
